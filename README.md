# Alignment Crest Website

A professional website for Alignment Crest, a rent collection agency specializing in recovering rental income on behalf of property owners.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Custom cursor, scroll animations, and hover effects
- **Contact Form**: Functional contact form with validation
- **Statistics**: Animated counters showing company achievements
- **Testimonials**: Customer testimonials and reviews
- **Services Overview**: Detailed description of collection services
- **Process Explanation**: Step-by-step guide to the recovery process

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework
- **GSAP**: Animation library for smooth interactions
- **Google Fonts**: Cormorant Garamond and Inter fonts
- **Unsplash Images**: High-quality stock photos

## Sections

1. **Hero Section**: Compelling headline with call-to-action
2. **Statistics**: Key performance metrics
3. **Services**: Three main service offerings
4. **Process**: Four-step recovery process
5. **Testimonials**: Client success stories
6. **Contact**: Contact form and company information
7. **Footer**: Additional links and social media

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. The website is fully self-contained with no build process required

## Browser Support

- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## Customization

The website uses external CDNs for fonts and libraries. For production use, consider hosting these assets locally.
